.item

Есть овверрайд на десктопы и мобилки

Это одна из двух версий айтема игры. Есть еще .unit. Так вот .item - это "легкая" версия. Тут простая разметка и нет никаких проблем вокруг позиционирования картинки (box art). В подвале нет иконок обмена или валют, только текст.

.item используется в версии для мобилок и при небольших ширинах на десктопе. Соответствено, он глобален. .unit же используется только на десктопах и только на больших ширинах. Вообще .item и .unit - это вещи взаимоисключающие.

Для позиционирования .item`ов (для создания плитки-водопада) используется блок и одноименный плагин .bricklayer. Для .unit`ов используется .unit.

Окончательно получается, что у нас есть две пары для коробок игр: .item и .bricklayer, .unit и .tiles.

Ретинизируем через srcset, просто подсовываем вторую картинку для @2x. дефолтная измеряется от ширины (ширина 160px, высота любая), ретинизированная, соответствено 320px ширины.

При желании в srcset можно добавлять картинки для @1.5x и для @3x. Таких телефонов уже достаточно много. Вопрос в месте на хостинге.
